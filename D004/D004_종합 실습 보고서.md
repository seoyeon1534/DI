# 데이터 분석 보고서
---
## 1. 분석 개요
- 대상: 모두마켓 주문 1,500건 (2025-01 ~ 2025-05), 고객·상품 정보 병합
- 방법: 3개 테이블 키 검증 후 m:1 병합 → 월별 KPI 집계 → 교차표·점유율 분석

## 2. 핵심 발견 (위 셀에서 추출한 숫자로 채우세요)
- 매출이 가장 높았던 달은 {3월 : 20035000 } 였으며, 전월 대비 증감률은 매출증감률(%) 열에서 17.6%로 확인할 수 있었음
- 카테고리 중 {패션}, 지역 중 {인천} 의 매출 기여가 가장 컸음.
- 전체 객단가는 약 {60,637}원.

### 2-1 해석
- 3월 매출 증가는 학생들의 새학기 시즌과 계절 변화에 따른 소비 증가의 영향으로, 관련 상품 수요가 확대되어 매출이 증가한 것으로 판단된다.

## 3. 데이터 신뢰성 (반드시 기록)
- 병합 전 키 검증: 중복 0건, 미매칭 0건 → 행 폭증·조용한 결측 없음.
- validate="m:1" 통과 → 관계 가정(주문:상품 = m:1) 유효.

## 4. 제언
- 월별 매출 변동의 원인(고객 수 ↑ vs 객단가 ↑)을 구매고객수·객단가 열로 분해해 마케팅 전략 차별화.
- 3월 매출 증가가 새학기 및 계절성 수요의 영향으로 판단되는 만큼, 성수기 이전 관련 상품의 재고 확보와 프로모션을 강화할 필요가 있다고 판단
